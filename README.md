🤖 Hello-Bot (Machine Learning Project)

[Predict student marks based on study hours , sleep hours, and dedication level using Machine Learning]


---

🚀 Features

Machine Learning Model: Model ko train karne ke liye train.py ka use kiya gaya hai.

Saved Model: Pre-trained model model.pkl file me saved hai taaki baar-baar train na karna pade.

Interactive UI: User-friendly frontend/backend setup (app.py aur frontend.py).



---

📂 Project Structure

Hello-Bot/

├── model.pkl       # Saved Machine Learning model
